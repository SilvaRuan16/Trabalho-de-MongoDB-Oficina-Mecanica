# Trabalho Produto 1 - Oficina Mecânica

Este repositório contém o desenvolvimento do Trabalho Produto 1 da disciplina, com o tema Oficina Mecânica, referente ao 5 período. O objetivo do trabalho é utilizar recursos do MongoDB para modelar e manipular dados relacionados ao funcionamento de uma oficina mecânica.

O projeto envolve o cadastro de veículos e serviços, representando a relação entre essas collections dentro do banco de dados. Além disso, o trabalho propõe a implementação de operações de CRUD utilizando propriedades e comandos do MongoDB, permitindo criar, consultar, atualizar e remover documentos conforme as necessidades do sistema.

Os dados utilizados no trabalho foram organizados em collections como `veiculos` e `servicos`, simulando situações comuns de uma oficina, como cadastro de automóveis, registro de serviços realizados e associação de vários serviços a um mesmo veículo.
